# Bold Innovation Landing Page

A modern landing page for Bold Innovation built with Next.js, React, and shadcn/ui.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components including shadcn/ui components
- `lib/` - Utility functions
- `.github/workflows/` - GitHub Actions workflows for CI/CD

## Technologies

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vercel (for deployment)

## Deployment

The project is configured for deployment to Vercel via GitHub Actions. The workflow file is located at `.github/workflows/frontend_prod.yml`.
